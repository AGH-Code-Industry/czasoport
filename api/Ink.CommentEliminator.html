Class CommentEliminator

Namespace
Ink

Pre-pass before main ink parser runs. It actually performs two main tasks:

  • comment elimination to simplify the parse rules in the main parser
  • Conversion of Windows line endings (\r\n) to the simpler Unix style (\n), so we don't have to worry about them later.
